其中三个必须的参数为：

选项 默认值 描述
id 必须值 歌曲 id / 播放列表 id / 相册 id / 搜索关键字
server 必须值 音乐平台: netease, tencent, kugou, xiami, baidu
type 必须值 song, playlist, album, search, artist

参数解释：

id=’外链播放器id’,必须参数,Id
type=[song=单曲, playlist=歌单, album=专辑, search=搜索结果, artist=艺术家],必须参数,调用类型
server=[netease=网易云音乐, tencent=QQ音乐, kugou=酷狗音乐, xiami=虾米音乐, baidu=百度音乐]

替换为自己的歌单

打开QQ音乐网页版，找到我的歌单，点击分享->复制链接，将该链接复制到任意文本编辑器中，例如可以看到如下链接：

https://i.y.qq.com/n2/m/share/details/taoge.html?id=9495604772

其中的 id 就是歌单 id
