hexo + Typora 简易化配置

大部分情况下我们是从外部直接复制图片粘贴进来，这时候图片的路径是指向本地默认存储路径的，如果我们希望图片粘贴进来后能够自动完成如下操作：

  • 自动复制到 ./文档名/<图片名>.png
  • 直接以 ![](<图片名>) 的形式展示

调整完毕后，当我们通过 ctrl + c 复制图片到 Typora 文档，就会自动将图片自动复制到 ./文档名/<图片名>.png
